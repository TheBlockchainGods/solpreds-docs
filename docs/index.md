## üéØ How It Works

1. **Connect Wallet** - Use Phantom, Backpack, or any Solana wallet
2. **Choose Direction** - Bet UP or DOWN on the next 5-minute round
3. **Watch Price** - Round locks, price captured from Pyth oracle
4. **Claim Winnings** - Winners split the pool proportionally
